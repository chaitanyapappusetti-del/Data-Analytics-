# ApexPlanet Data Analytics - Task 1

## Data Cleaning and Preprocessing using Python

### Objective
The objective of this task is to load, inspect, clean, and preprocess the dataset using Python and Pandas.

### Tools Used
- Python
- Google Colab
- Pandas

### Dataset
- ApexPlanet_DataAnalytics_Dataset.xlsx

### Steps Performed
- Read the Excel dataset.
- Displayed the first five rows.
- Checked dataset information.
- Generated summary statistics.
- Checked for missing values.
- Identified and removed duplicate records.
- Filled missing values using forward fill.
- Converted the `Order_Date` column to datetime format.
- Created an `Age_Group` column.
- Saved the cleaned dataset as a new Excel file.

### Output
- Cleaned dataset
- Dataset information
- Statistical summary
- Missing value report
- Duplicate record report

### Conclusion
The dataset was successfully cleaned and preprocessed, making it ready for further analysis and visualization.
